                            To define table actions on specific elements, handles can be used.
                            A handle is given a special attribute :
                            
                          

                          
    
                            
  • [coral-table-select] - Select/unselect all table items.
    • 
                            
  • [coral-table-rowselect] - Select/unselect the table item.
    • 
                            
  • [coral-table-roworder] - Drag to order the table item.
    • 
                            
  • [coral-table-rowlock] - Lock/unlock the table item.

Custom sorting

                  

                    If setting sortabletype to custom, columns won't sort based on the default table sorting.
                    Instead, a custom sorting can be performed when triggered by user interaction. This can be defined by listening to
                    the coral-table:beforecolumnsort event.
