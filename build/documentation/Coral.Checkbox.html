Coral.Checkbox

Current Version: 1.0.2
Related Components: Coral.Radio, Coral.ButtonGroup, Coral.mixin.formField
Github: coralui-component-checkbox

This component follows the Coral.mixin.formField specification. Please keep in mind that readOnly and invalid will be ignored, just like in a native HTMLInputElement.

instance.remove

    

    






    Non-destructively remove this element. It can be re-added by simply appending it to the document again.
    It will be garbage collected if there are no more references to it.
    This is different from using jQuery's remove(), which is destructive in that it removes data and listeners.
