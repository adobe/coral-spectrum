Coral.Switch

Current Version: 3.0.2
Related Components: Coral.Checkbox, Coral.ButtonGroup
Github: coralui-component-switch

A Switch is a representation of a mutually exclusive where it is implied they one activates and the other one disables the option. Since it does not contain a label, it should not be used to select between to options that do not share this Yes/No implication. This is similar to a checkbox and is often used in tables.

Accessibility

              

                      
Keyboard support

                  

                    Coral.Switch can receive keyboard focus and its state can be changed via SPACE key, and is identified
                    to assistive technology as a checkbox element.
                    
                  

                      
Labelling the component

                  

                    To meet accesibility standards, the switch component should always be labelled. This can be done by setting the
                    labelledby attribute with the id of the HTML element that provides a label for the Switch.
