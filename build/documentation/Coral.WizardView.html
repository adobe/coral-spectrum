Coral.WizardView

Current Version: 1.0.0
Related Components: Coral.StepList, Coral.PanelStack
Github: coralui-component-wizardview

Coral.WizardView API

Constructor

JavaScript:

new Coral.WizardView() or document.createElement('coral-wizardview')

HTML Tag:

<coral-wizardview>

Extends

  • Coral.Component

Instance Properties

instance.panelStacks {Coral.Collection}

Readonly
The set of controlled PanelStacks. Each PanelStack must have the coral-wizardview-panelstack attribute. See Coral.Collection for more details regarding collection APIs.

instance.stepLists {Coral.Collection}

Readonly
The set of controlling StepLists. Each StepList must have the coral-wizardview-steplist attribute. See Coral.Collection for more details regarding collection APIs.
